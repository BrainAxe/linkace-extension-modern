# Icons Directory

This directory should contain the extension icons in PNG format:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Temporary Solution

For development, you can:

1. Copy icons from the old extension: `/home/rizwan/Public/Yet-Another-LinkAce-Extension-dev-vue2/public/icons/`
2. Or create simple icons using an online tool like:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

## Creating Icons

Design guidelines:
- Use a simple, recognizable symbol (like a link chain or bookmark)
- Use the LinkAce brand colors if available
- Ensure good contrast for visibility
- Export in PNG format with transparency
- Test how they look in light and dark browser themes
